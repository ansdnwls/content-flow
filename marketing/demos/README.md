# Demo Assets

This folder contains capture-ready scripts and storyboards for launch media.

Included:

- `demo-storyboard.md` for the 30-second GIF
- `explainer-script.md` for the 60-second walkthrough
- `capture-checklist.md` for clean screen recording

Note:

Binary GIF and MP4 exports are not committed here. The repository stores the source scripts and storyboard so the media can be recorded consistently from the live product.
