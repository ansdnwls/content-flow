# Changelog

This page mirrors the repository `CHANGELOG.md` in a docs-friendly format.

## 0.2.0 · 2026-04-07

### Added

- Webhook retry queue, dead-letter handling, signed delivery verification helpers, and replay APIs
- Automatic OAuth token refresh worker plus `account.disconnected` webhook dispatch
- Platform-aware publish throttling for YouTube, TikTok, Instagram, X, and LinkedIn
- Structured logging, request IDs, metrics, and expanded health endpoints
- Additional adapters and tests for LINE, Mastodon, and Medium
- CI workflows for releases, SDK publishing, container publishing, and deployment health checks

### Changed

- Unified release metadata at `0.2.0` across API and SDK artifacts
- Expanded readiness checks for Supabase, Redis, and Celery
- Automated Go SDK tagging for `pkg.go.dev`

### Verification

- `ruff check .`
- `pytest -q`
- `pytest --cov=app --cov-fail-under=80`

## 0.1.0 · 2026-04-07

### Added

- Posts, videos, bombs, comments, predict, schedules, usage, accounts, and analytics APIs
- 18-platform publishing baseline with regional support such as Naver Blog, Tistory, Kakao, and note.com
- OAuth integrations for Google, Meta, TikTok, and X
- Encrypted token storage, video generation, analytics snapshots, billing limits, and rate limiting
- Landing site, SDKs, Docker stack, and OpenAPI export tooling

### Fixed

- Remaining `ruff` cleanup
- Video-to-post publish wiring
- Encrypted OAuth token loading in publish flows
- Usage history aggregation regressions
