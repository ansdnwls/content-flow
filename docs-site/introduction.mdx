# Build once. Distribute everywhere.

ContentFlow exposes one API for:

- Multi-platform post creation and scheduling
- AI video generation with optional auto-publish
- Content Bomb expansion from one topic to many channel variants
- Comment collection and AI-assisted replies
- Webhook delivery replay and dead-letter recovery
- Analytics, trending, billing, and workspace operations

## Base URL

```txt
https://contentflow-api.railway.app
```

## Product surface

| Area | Core endpoints |
| --- | --- |
| Posts | `POST /api/v1/posts`, `GET /api/v1/posts`, `DELETE /api/v1/posts/{id}` |
| Accounts | `POST /api/v1/accounts/connect/{platform}`, `GET /api/v1/accounts` |
| Videos | `POST /api/v1/videos/generate`, `GET /api/v1/videos/{id}` |
| Webhooks | `GET /api/v1/webhooks/dead-letters`, `POST /api/v1/webhooks/{webhook_id}/redeliver` |
| Analytics | `GET /api/v1/analytics*` endpoints in the generated reference |
| Privacy & consent | `POST /api/v1/consent/grant`, `POST /api/v1/privacy/export` |
