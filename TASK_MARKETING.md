> 담당: 디자인팀
> 우선순위: P1 (출시 임팩트 극대화)

---

## 배경

브랜드 아이덴티티와 디자인 시스템을 잡았다. 이제 그 시스템을 활용해서 **사람들이 보고 공유하고 싶어지는 마케팅 자산**을 만든다.

ContentFlow의 출시 순간 SNS, Product Hunt, Hacker News에서 어떻게 보이는지가 초기 트래픽을 결정한다.

이 작업은 **"한 번 보면 클릭하지 않을 수 없는 비주얼"** 만들기다.

## 작업 2: 인터랙티브 데모 (랜딩페이지 핵심)

### 파일: `landing/components/interactive-demo.tsx`

랜딩페이지 hero 아래 또는 별도 섹션에:

**"실제로 작동하는 척하는 데모"**

사용자가 폼 채우면:
1. "주제를 입력하세요" → 사용자 타이핑
2. "플랫폼 선택" → 21개 플랫폼 카드 (다중 선택)
3. "Generate" 클릭
4. **시각적으로 폭발**:
   - AI 영상 생성 진행바 (페이크)
   - 21개 플랫폼에 동시 발행되는 애니메이션
   - 각 플랫폼 아이콘이 ✓로 변하는 애니메이션
   - 마지막에 "Sign up to do this for real" CTA

데이터는 mock이지만 시각적으로 임팩트 있게.

### 변형: 코드로 보여주기
- 같은 화면에 코드 탭
- 사용자가 폼 입력하면 코드도 자동 업데이트
- "보세요, API 호출 한 번이에요"

## 작업 3: 이메일 템플릿 비주얼 업그레이드

### 디렉토리: `app/templates/emails/`

기존 10개 이메일 템플릿을 브랜드 시스템에 맞게 재디자인:

### 공통 디자인
- 헤더: 로고 + 그라디언트 배경
- 본문: 화이트 카드 (다크 배경 위)
- CTA 버튼: 그라디언트 + 호버 효과 (이메일 클라이언트 호환)
- 푸터: 소셜 링크 + unsubscribe

### 각 템플릿 특화
- **welcome.html**: 거대한 환영 비주얼, 플랫폼 그리드
- **payment_succeeded.html**: 영수증 디자인, 플랜 배지
- **payment_failed.html**: 빨간 액센트, 명확한 액션 버튼
- **monthly_summary.html**: 통계 시각화 (인포그래픽 스타일)
- **account_disconnected.html**: 경고 톤, 재연결 버튼 강조

이메일은 인라인 CSS만 가능. 호환성 테스트 필수 (Gmail, Outlook, Apple Mail).

## 작업 4: 에러 페이지 + 빈 상태

### 파일: `dashboard/app/error.tsx`, `dashboard/app/not-found.tsx`

### 404 페이지
- 큰 "404" 타이포그래피
- 재미있는 일러스트 (브랜드 톤)
- "홈으로" + "이전 페이지" 버튼
- ContentFlow 위트 (예: "이 페이지는 21개 플랫폼 중 어디에도 없네요")

### 500 페이지
- 진정시키는 메시지
- "다시 시도" 버튼
- 상태 페이지 링크

### 403 (권한 없음)
- "이 페이지는 당신을 위한 게 아니에요"
- 로그인 또는 업그레이드 CTA

### 유지보수 페이지
- "잠깐 멈춰있어요"
- 복귀 예상 시간
- 트위터/디스코드 링크

### 빈 상태 일러스트
- 빈 포스트 목록 → "첫 포스트를 만들어보세요"
- 빈 분석 → "데이터가 모이는 중"
- 빈 계정 → "계정을 연결해보세요"

각 빈 상태에 명확한 다음 액션 버튼.

## 의미

지금까지 만든 게 **제품**이라면, 이번 작업은 **포장**이다.

같은 제품이라도:
- 흔한 SaaS 포장 → 무관심
- 임팩트 있는 포장 → "이거 뭔데?" → 클릭 → 가입

Product Hunt 출시일에 사람들이 처음 보는 게 썸네일과 gallery 이미지다. 그게 좋으면 트래픽이 폭발하고, 평범하면 묻힌다.

"포장이 본질보다 중요하다"는 말이 아니다. **포장도 본질의 일부**다. ContentFlow의 본질이 "21개 플랫폼을 1 API로"라면, 그 메시지가 5초 안에 시각적으로 전달돼야 한다.

이 작업이 ContentFlow 출시일의 첫 100명 사용자를 결정한다.
