# ContentFlow Brand

## Positioning

ContentFlow is the publishing engine for creators and content teams that care more about shipping velocity than dashboard ceremony.

Primary line:

**The API for creators who ship.**

Supporting lines:

- One topic. 21 platforms. Zero clicks.
- Stop posting. Start shipping.
- The unfair API for content.

## Recommended Direction

### Chosen Palette: Electric Sunset

This is the strongest fit because ContentFlow is not a finance dashboard or infra console. It is an orchestration product for creators, operators, and media teams. The visual system should feel like a burst of distribution energy, not a terminal clone.

- Primary: `#FF6B35`
- Secondary: `#FFD23F`
- Accent: `#06FFA5`
- Dark Background: `#120A14`

Why this works:

- Orange makes the product feel fast, catalytic, and output-oriented.
- Yellow carries amplification and visibility.
- Mint introduces a technical edge without turning the whole brand into cyberpunk cliche.
- The deep ink background keeps the system premium and lets the bright colors read as controlled signal, not noise.

## Typography

Recommended pair:

- Display: `Bricolage Grotesque`
- Body: `Manrope`
- Mono: `IBM Plex Mono`

Rationale:

- `Bricolage Grotesque` has enough character to avoid template-SaaS sameness.
- `Manrope` stays clean and readable across dense product UI.
- `IBM Plex Mono` keeps code, metrics, and API references sharp.

Scale:

- `12` xs
- `14` sm
- `16` md
- `20` lg
- `24` xl
- `32` 2xl
- `48` display-sm
- `64` display-md
- `96` display-lg

Line height:

- Dense display: `0.92` to `1.0`
- Standard UI text: `1.45` to `1.6`
- Long-form copy: `1.7`

Letter spacing:

- Display: `-0.06em` to `-0.03em`
- UI titles: `-0.02em`
- Eyebrows / labels: `0.12em` to `0.18em`

## Logo System

Concept:

The mark is a central node that explodes into multiple endpoints. It visualizes one input becoming many platform outputs.

System assets:

- `design/logos/contentflow-wordmark-dark.svg`
- `design/logos/contentflow-wordmark-light.svg`
- `design/logos/contentflow-symbol-dark.svg`
- `design/logos/contentflow-symbol-light.svg`
- `design/logos/contentflow-favicon.svg`

Usage:

- Symbol for app icon, favicon, and condensed sidebar marks
- Wordmark for landing header, auth screens, and documentation
- Use the burst motif sparingly; it is the signature, not a wallpaper pattern

## Signature Visual Rules

- Avoid purple-led gradients entirely.
- Large type should feel compressed, vertical, and slightly aggressive.
- Use bright accents on mostly dark surfaces with generous negative space.
- Motion should imply signal propagation: pulse, fan-out, scan, and draw-on.
- Platform surfaces should feel like cards in a distribution network, not social logos pasted on a grid.
