# 카드뉴스 디자인 가이드

## 기본 구조 (3파트)

| 파트 | 장수 | 역할 | 핵심 요소 |
|------|------|------|----------|
| 표지 | 1장 | 첫인상, 클릭 유도 | 메인 타이틀 + 서브 타이틀 + 임팩트 있는 배경 |
| 본문 | 2~10장 | 정보 전달 | 한 장 = 하나의 핵심 메시지 + 제목 + 내용 + 이미지 |
| 마무리 | 1장 | 행동 유도(CTA) | 요약 or 다음 행동 안내 + 브랜드/출처 |

## 유형 5가지

| 유형 | 적합한 콘텐츠 |
|------|-------------|
| 나열형(리스트형) | "~하는 5가지 방법", 팁 모음 |
| 스토리텔링형 | 브랜드 스토리, 사례 소개 |
| 집중형 | 어려운 개념 설명, 교육형 |
| 문답형(퀴즈형) | 상식, Q&A |
| 웹툰형/오토형 | 공감형, 재미 위주 |

## 스토리라인 공식

공식 A — 스토리텔링형 (설득): WHY → HOW → WHAT

공식 B — 집중형 (교육): WHAT(핵심) → WHY → HOW → WHAT(요약)

## 디자인 규칙

- 색상: 한 페이지 2~3색 제한, 통일된 팔레트
- 폰트: 본문 고딕 계열 (노토 산스, 나눔 고딕), 꾸밈 폰트는 제목에만
- 행간: 폰트 크기 × 1.5~1.6 = 행간
- 여백: 충분한 여백 확보 (부족하면 가독성 급락)
- 텍스트 효과: 그림자/라인 금지 — 단순한 글자가 전달력 높음
- 텍스트 배치: 긴 문장은 왼쪽 정렬, 짧은 문구만 중앙 정렬
- 페이지 표시: 전체 몇 장인지 하단에 표시 (이탈률 감소)
- **본문은 한 카드에 2~3줄 이내로 압축** (모바일 가독성)

## 모바일 가독성 폰트 사이즈 (1080px 캔버스 기준)

1080px 이미지가 모바일(~375px)에서 약 1/2.88로 축소된다. 모바일에서 읽히려면 캔버스 기준 충분한 크기가 필요하다.

| 요소 | 캔버스 최소 | 모바일 체감 |
|------|-----------|-----------|
| 제목 (h2) | **48px** | ~17px |
| 본문 (desc) | **36px** | ~12px |
| 날짜/태그 | **32px** | ~11px |
| 위치/출처 (info) | **30px** | ~10px |
| 페이지 번호 | **28px** | ~10px |
| **절대 최소** | **24px** | ~8px |

## 레이아웃별 페이지 번호 위치

- 풀블리드, 상하분할: 카드 하단 중앙
- **좌우분할: 텍스트 영역 하단 중앙** (이미지 위에 놓지 않는다)

## 사이즈

| 비율 | 픽셀 | 용도 |
|------|------|------|
| 4:5 (세로형) | 1080 x 1350px | 인스타그램 권장 (2025년~) |
| 1:1 (정사각형) | 1080 x 1080px | 기본형 |
| 안전 영역 | 중앙 1080x1080px | 세로형 사용 시 핵심 요소 배치 |

## 블랙앤화이트 에디토리얼 테마 (이 스킬 기본값)

| 항목 | 설정 |
|------|------|
| 배경 | #0A0A0A (딥블랙) 또는 #FAFAFA (오프화이트) 교차 |
| 텍스트 | 대비 극대화, 타이포그래피 중심 |
| 폰트 | 디스플레이용 세리프 + 본문용 산세리프 |
| 레이아웃 | 비대칭 구도, 이미지 블리드, 넉넉한 여백 |
| 이미지 | AI 생성 이미지를 히어로 이미지로 대담하게 배치 (카드의 60%+) |

## 기획서 출력 포맷

기획서는 아래 구조로 작성한다:

1. 유형 선택 (나열형/스토리텔링형 등) + 선택 이유
2. 총 장수
3. 장별 테이블: 장번호 | 파트(표지/본문/마무리) | 핵심 메시지 | 필요한 이미지 설명
4. 디자인 스펙 요약
